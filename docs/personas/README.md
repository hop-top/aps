# Personas

Actor personas and profile persona configuration for the APS system.

## 👤 Actor Personas

These represent the different types of users who interact with APS.

| Persona | Description | Stories |
|---------|-------------|---------|
| [User](user.md) | End-user who creates profiles, runs commands, and manages capabilities | 20 stories |
| [Maintainer](maintainer.md) | Developer who writes and runs automated E2E tests | 4 stories |
| [External Client](external-client.md) | API consumer interacting via HTTP protocol adapters | 5 stories |
