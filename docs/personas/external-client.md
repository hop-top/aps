# External Client

## Description

An external system or API consumer that interacts with APS through HTTP protocol adapters (Agent Protocol, OpenAI Assistants API, etc.). This persona cares about standard API compliance, real-time streaming, session management, and the ability to discover and invoke agents programmatically.
